# Cidadania Digital e Inteligência Artificial
**Objetivo do Site:** Conscientizar a comunidade escolar sobre os perigos das mídias manipuladas por IA (deepfakes) e ensinar boas práticas para identificar e combater a desinformação no ambiente digital.

**Uso de IA:** * Textos explicativos e estrutura de código base gerados com auxílio da IA Gemini.
* Prompt utilizado: "Crie um site educativo em HTML5, CSS3 e JavaScript Vanilla sobre Cidadania Digital e o impacto de Deepfakes".
